<decisions_made>

- Single main IPK + optional iStore meta package; do not ship separate OpenWrt vs iStoreOS app builds.
- Install app files under `/usr/lib/tg-downloader-ui`; create `/opt/tg-downloader-ui` symlink only when `/opt` is writable.
- Default download path is `/root/telegram-downloads` because `/mnt` is often a tiny tmpfs until external storage is mounted.
- Telethon proxy config is returned as a dict with credentials; bare `(scheme, host, port)` is incorrect for authenticated proxies.
- tdl is not bundled in the IPK; operators must install the upstream binary separately.

</decisions_made>
