<decisions_made>

- Preferred fix: use Linux `SIGSTOP/SIGCONT` for a genuine live pause because it preserves the tdl process, open file descriptor, and current byte offset.
- Keep `--continue` only for its actual purpose: skipping fully completed items after relaunch.
- Docker and OpenWrt should have identical pause behavior through shared application code.
- Container/application restart recovery for a partially downloaded single file remains unsupported by upstream tdl 0.20.3 unless tdl is patched or replaced.
